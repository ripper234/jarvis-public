# Style Guide

## Colors
- **Primary**: `#4f46e5` (indigo)
- **Primary hover**: `#6366f1`
- **Primary dark**: `#4338ca`
- **Accent gradient**: `#4f46e5 → #7c3aed → #06b6d4`
- **Background**: `#fafafa`
- **Text primary**: `#111`
- **Text secondary**: `#444`
- **Text muted**: `#888`
- **Text faint**: `#bbb`, `#ccc`
- **Borders**: `rgba(0,0,0,0.06)`
- **Tag active**: indigo on indigo-light
- **Tag launched**: `#10b981` on green-light
- **Tag ideation**: `#9ca3af` on gray-light

## Typography
- **Font stack**: Inter, -apple-system, sans-serif
- **Mono**: JetBrains Mono
- **H1**: 1.8rem / 700 / #111 / -0.03em tracking
- **H2**: 1.15rem / 600 / #111
- **H3**: 0.95rem / 600 / #222
- **Body**: 0.88rem / 1.7 line-height
- **Small**: 0.82rem, 0.78rem, 0.75rem, 0.68rem (decreasing emphasis)

## Spacing
- **Page max-width**: 800px
- **Page padding**: 2.5rem 1.25rem (desktop), 1.5rem 1rem (mobile)
- **Card padding**: 1.25rem (desktop), 1rem (mobile)
- **Card border-radius**: 14px
- **Grid gap**: 0.75rem
- **Section margin**: 2.5rem top

## Components
- **Cards**: glass effect (rgba white + backdrop-filter blur)
- **Tags**: pill shape (border-radius 20px), uppercase, 0.6rem
- **Callout/TLDR**: left border accent + light bg
- **Divider**: 1px #eee, 2.5rem margin
- **Back link**: 0.8rem, #999, hover → indigo
- **CTA button**: indigo bg, white text, 10px radius

## Decorative
- **Gradient orbs**: fixed position, blurred circles (indigo + emerald)
- **Top accent bar**: 3px gradient (indigo → purple → cyan)

## Breakpoints
- **Mobile**: 480px
- **Small mobile**: 320px
- **Grid collapse**: 2-col → 1-col at 320px, 3-col → 1-col at 480px
