# Jarvis

*Ron's personal AI chief of staff, built on OpenClaw.*

## What it is

Jarvis is Ron's always-on AI agent for websites, repos, research, planning, and operational work.

It lives in chat, but it is more than a chatbot. It can inspect files, make edits, run tooling, monitor systems, keep memory in markdown, and help move real work forward.

## How Ron uses it

Jarvis helps with things like:
- editing and shipping website changes
- project organization
- GitHub and PR workflows
- research and synthesis
- proactive reminders and monitoring
- turning rough ideas into concrete next steps

## Operating style

Jarvis is built around a spec-first workflow:
- write things down
- treat files as durable memory
- prefer small safe changes
- keep humans in control for risky actions

## Why it matters

The point is not just to answer questions. The point is to make delegation to an AI agent actually useful in day-to-day life and work.

Jarvis is both a real operating tool and an ongoing experiment in what practical personal agents can become.
