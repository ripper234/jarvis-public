🎓 Bot Academy — Deep Dive

Track how agents and humans discover, learn from, and reuse our lessons.

Conversion Funnel

Set up as Plausible Funnel with these goals:

1 Visit learn.html pageview: /learn.html
↓
2 Expand a section (engaged reader) event: section-expand
↓
3 Copy the agent prompt (high intent) event: prompt-copy
↓
4 Click external resource event: link-external

Tracked Events

  • section-expand — which sections get opened (props: section)
  • prompt-copy — agent prompt was copied (high-value signal)
  • link-external — clicked through to a resource (props: target = hostname)
  • scroll-depth — reached 25%, 50%, 75%, 90% of page
  • time-on-page — stayed 30s, 60s, 2m, 5m

Plausible Goals to Create

In Plausible → Site Settings → Goals, add:

  1. Custom event: section-expand
  2. Custom event: prompt-copy
  3. Custom event: link-external
  4. Custom event: scroll-depth
  5. Custom event: time-on-page
  6. Funnel: Visit → Expand → Copy → External click

Key Questions This Answers

  • Which sections do people actually read? (section-expand breakdown)
  • Are agents copying the prompt? (prompt-copy rate)
  • Where do visitors come from? (Plausible referrers)
  • Do people stay or bounce? (scroll-depth + time-on-page)
  • Which external resources get clicks? (link-external breakdown)

⚖️ Balance — Deep Dive

Track the path from landing page visit to beta signup via WhatsApp.

Conversion Funnel

Set up as Plausible Funnel:

1 Visit balance.html pageview: /balance.html
↓
2 Read the page (scroll ≥75%) event: scroll-depth {depth: "75%"}
↓
3 Click WhatsApp CTA event: cta-click {target: "wa.me/..."}
↓
4 Send WhatsApp message (manual CRM entry) tracked in Balance CRM

Tracked Events

  • cta-click — WhatsApp CTA button clicked (the money event)
  • scroll-depth — how far visitors read (25%, 50%, 75%, 90%)
  • time-on-page — engagement duration (30s, 60s, 2m, 5m)
  • link-external — any outbound link clicks

Clarity Heatmap Priorities

In Clarity, filter to /balance.html and check:

  • Click heatmap: Where do people click? Is the CTA getting attention?
  • Scroll heatmap: What % of visitors see the CTA? (it needs to be above the fold or within natural scroll)
  • Session recordings: Watch 5-10 sessions to see hesitation patterns before CTA click
  • Dead clicks: Are people clicking non-interactive elements? (opportunity for more content)

Key Questions This Answers

  • What % of visitors click the WhatsApp CTA? (cta-click / pageviews)
  • Do people read the full page or bounce early? (scroll-depth distribution)
  • Where do beta testers come from? (Plausible referrers for /balance.html)
  • Is the page too long/short? (Clarity scroll heatmap)
  • What's the CTA click → actual WhatsApp message conversion? (CRM cross-reference)

Optimization Ideas (data-driven, run after 100+ visitors)

  • If scroll-depth drops before CTA: move CTA higher or add above-the-fold CTA
  • If CTA clicks high but WhatsApp messages low: simplify the pre-filled message
  • If time-on-page <15s: add engaging content (video, testimonials, feature preview)
  • A/B test CTA copy: "Join the Beta →" vs "Message me on WhatsApp →"
  • Add UTM parameters to all distribution links for referrer attribution
