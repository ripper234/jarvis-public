Methodology
The principles behind spec-first AI development
🧠
Agent as Co-Founder

Not a tool you prompt. A partner that reads specifications, understands intent, and proposes tiny improvements. 1-2 lines per PR. The agent thinks, the human approves.

🎯
Attention as Currency

10 free + 20 earnable credits daily. Capture mode costs zero -- fire off ideas via voice, the agent files everything. Only the morning summary costs attention.

🔒
Safety by Architecture

No direct write access to anything. Every mutation goes through a code-level guard. Rules enforced by scripts, not prompts the AI might forget. Trust is earned through structure.

📄
Spec is the Product

Top-down: spec first, then tests, then architecture, then code. Most of the code doesn't exist yet -- and that's intentional. The spec IS the product.
